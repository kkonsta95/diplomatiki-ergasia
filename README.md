# diplomatiki-ergasia


Σε αυτό το κεφάλαιο θα δοθεί ένας ορισμός των ειδικών μορφών τουρισμού. Η αναζήτηση διαφορετικών εμπειριών από τους τουρίστες ωθεί στην ανάπτυξη νέων εξειδικευμένων μορφών τουρισμού.

2.5. Κίνητρα

Σε αυτό το κεφάλαιο θα εξεταστούν τα κίνητρα των τουριστών αυτών, ποιοι είναι αυτοί οι τουρίστες, με βάση την τυπολογία τους, και τι αναζητούν. Οι τουρίστες αυτοί επιδιώκουν, φυσικά, να αναζητήσουν νέες εμπειρίες, νέα συναισθήματα και συγκινήσεις για τα οποία είναι διατεθειμένοι να "πληρώσουν". Η ιδιαιτερότητα αυτού του είδους του τουρίστα έγκειται στο γεγονός ότι δε στοχεύει στα προιόντα αλλά στην ποιότητα των εμπειριών και των υπηρεσιών που του προσφέρονται. https://www-sciencedirect-com.proxy.eap.gr/science/article/pii/S0261517704002213
