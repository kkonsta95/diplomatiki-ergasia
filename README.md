# diplomatiki-ergasia


Σε αυτό το κεφάλαιο θα δοθεί ένας ορισμός των ειδικών μορφών τουρισμού. Η αναζήτηση διαφορετικών εμπειριών από τους τουρίστες ωθεί στην ανάπτυξη νέων εξειδικευμένων μορφών τουρισμού. Πρόκειται για μια διαφοροποιημένη μορφή τουρισμού, η οποία αντιτίθεται σθεναρά στο μαζικό τουρισμό. Ο τουρισμός ειδικών ενδιαφερόντων έχει ως στόχο να επενδύσει σε νέες μορφές τουρισμού, προσαρμοσμένες στις ανάγκες των τουριστών, όπως είναι ο αγροτουρισμός, ο πολιτιστικός τουρισμός, και ο τουρισμός εκδηλώσεων. Ένας άλλος ορισμός θα μπορούσε να είναι "η παροχή εξειδικευμένων υπηρεσιών ψυχαγωγίας καθοδηγούμενη από συγκεκριμένο ενδιαφέρον ατόμων ή ομάδων".

2.5. Κίνητρα

Σε αυτό το κεφάλαιο θα εξεταστούν τα κίνητρα των τουριστών αυτών, ποιοι είναι αυτοί οι τουρίστες, με βάση την τυπολογία τους, και τι αναζητούν. Οι τουρίστες αυτοί επιδιώκουν, φυσικά, να αναζητήσουν νέες εμπειρίες, νέα συναισθήματα και συγκινήσεις για τα οποία είναι διατεθειμένοι να "πληρώσουν". Η ιδιαιτερότητα αυτού του είδους του τουρίστα έγκειται στο γεγονός ότι δε στοχεύει στα προιόντα αλλά στην ποιότητα των εμπειριών και των υπηρεσιών που του προσφέρονται. Στο πλαίσιο αυτό έχει αναδυθεί μια πληθώρα από νέους όρους, που καθορίζουν τις τουριστικές τάσεις, όπως "εναλλακτικός", "βιώσιμος", και "τουρισμός ειδικών ενδιαφερόντων". https://www-sciencedirect-com.proxy.eap.gr/science/article/pii/S0261517704002213
